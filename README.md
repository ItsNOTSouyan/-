产品功能
1.主核心功能——药胡啦
1.1功能概述
药胡啦是一款结合中药方剂规则的麻将游戏。通过将传统麻将游戏与中药文化相结合，药胡啦为用户提供了一个有趣、富有教育意义的学习平台，让用户能够在游戏中学习中药方剂知识，感受中药文化的魅力。
<img width="418" height="229" alt="image" src="https://github.com/user-attachments/assets/45ee7ae6-2ec7-4e78-ae6b-9ea570a69bc4" />
1.2游戏规则
药胡啦的游戏规则基于传统麻将游戏进行改编，融入了中药方剂的知识元素。在游戏中，每张麻将牌都代表一味中药。玩家需要通过摸牌、碰杠、胡牌等麻将游戏的基本操作，组合出符合中药方剂规则的牌型。

2.其他核心功能——药韵
2.1功能概述
药韵是中药科普功能，包含了采药、炮制、配伍和药膳四个子版块。药韵功能旨在通过游戏化和图文并茂科普的方式，向用户普及中药知识，提高用户对中药文化的认识和兴趣。

2.2采药、炮制、配伍和药膳
（1）采药
①游戏形式
采药小游戏以“找你妹”为基础，游戏一局总共100秒，玩家需要在规定的时间内根据页面右边的卡片中药的外观，寻找并采集指定的中药。
<img width="1440" height="900" alt="采药2" src="https://github.com/user-attachments/assets/b61c454a-4922-4bc8-8f83-fcd0c83955c1" />

（2）炮制
采用横向画卷式的布局，按顺序展示中药炮制的各个环节，每个步骤都配图片和文字介绍，内容是中药起源、中药制作、修制、水制、火制、水火共制。
<img width="8640" height="900" alt="炮制页面" src="https://github.com/user-attachments/assets/a689b247-ecb8-4585-937e-5c2b64e7ad1e" />

（3）配伍
配伍小游戏以跳棋为基础，玩家需要移动棋子，同时根据中药配伍的原则“君臣佐使”，选择合适的中药进行搭配。向用户传递“君臣佐使”的中药配伍原则，并让用户了解中药如何搭配使用。
<img width="1440" height="900" alt="配伍" src="https://github.com/user-attachments/assets/32cc7697-20f3-4912-9d16-7cd03820fee5" />

（4）药膳
药膳是中药食疗的推荐，按春夏秋冬四个季节来推荐对应食谱。以瀑布流的布局和图文形式科普，向用户展示中药在日常生活中的运用。通过药膳的学习和实践，用户可以更好地了解中药的养生功效。
